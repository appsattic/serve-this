A small static web server to hoist static files onto a web port.

## Synopsis ##

```
$ git clone git://github.com/appsattic/serve-this.git
$ cd serve-this
$ npm -d install
$ ./serve-this.js path/to/your/htdocs
```

Now go to http://localhost:8080/ to see your static site.
